# Red Scanner con Python y Scapy

Este proyecto es un escáner de red, usa Python para detectar las ip activas en una red local.

## Características

- Escanea un rango de IP especificado en la red local.
- Fácil de usar y configurar.

## Requisitos

- Python 3.x
- Scapy
- Npcap (para Windows)
